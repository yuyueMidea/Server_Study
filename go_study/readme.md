go语言
- Go（Golang） 是 Google 开发的现代编程语言
- 强调：简洁、高性能、并发支持
- 适合构建：高性能 Web 后端、微服务、工具程序、CLI、网络服务

go的核心特点：
| 特点    | 说明                                           |
| ----- | -------------------------------------------- |
| 编译型语言 | 编译为机器码，启动速度快                                 |
| 静态类型  | 类型检查明确，编译时发现错误                               |
| 内存安全  | 自动垃圾回收                                       |
| 并发支持强 | 内置 `goroutine` 和 `channel`                   |
| 语法简洁  | 类似 C，但没有头文件和类                                |
| 内置工具链 | 格式化（`gofmt`）、构建（`go build`）、测试（`go test`）都自带 |
